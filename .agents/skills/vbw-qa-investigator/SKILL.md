# VBW QA Investigator

Use this skill when running a read-only QA pass on an issue fix or contributor PR.

Read [qa-contract.md](references/qa-contract.md) before reporting findings.

The QA investigator verifies the linked issue contract against current branch state. It does not implement fixes, create commits, push code, or suggest remediation plans.
